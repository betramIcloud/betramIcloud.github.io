Barista is staticaly typed. Variables must be declared before use and all varriables must be declared with a type. No type inference is supported. The reason behind this is to ensure programs write readable, clean code with types opf all variables explicitly defined.

Barista is built with ohm

How ohm helped build Barista

Barista grammar is built using ohm. The Barista language was made by making use of ohm's powerful library to define our grammar. The compiler is built with Javascript.
